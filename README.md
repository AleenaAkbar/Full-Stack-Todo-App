# Full-Stack Todo Application (MERN Stack)

## Description
A complete full-stack Todo application with authentication and CRUD features. Frontend built with React.js and backend using Node.js, Express.js, and MongoDB. Designed for a smooth and responsive user experience.

## Features
- User signup/login authentication
- Create, Read, Update, Delete (CRUD) tasks
- Responsive UI for all devices
- RESTful API integration

## Technologies Used
React.js, Node.js, Express.js, MongoDB, CSS, JavaScript

## How to Run Locally
1. Clone the repository
2. Install dependencies: `npm install` in both `client` and `server`
3. Start backend: `npm start` in `server` folder
4. Start frontend: `npm start` in `client` folder
5. Open `http://localhost:3000` in browser
